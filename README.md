📅 Interactive Calendar

A modern and interactive calendar application built using React + Vite. This project provides a clean UI with smooth navigation and a responsive layout, inspired by real-world wall calendar designs.

🚀 Features
📆 Monthly calendar view
🔄 Navigate between months
📝 Add and manage notes/events (if implemented)
🎨 Clean and responsive UI
⚡ Fast performance using Vite
🛠️ Tech Stack
Frontend: React
Build Tool: Vite
Styling: CSS / Tailwind (update if used)

⚙️ Installation & Setup
Clone the repository:
git clone https://github.com/aditiisingh0/Interactive-Calendar-.git
Navigate to the project folder:
cd Interactive-Calendar-
Install dependencies:
npm install
Run the development server:
npm run dev
